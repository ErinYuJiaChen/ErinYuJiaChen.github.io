Vocab Definition
Protocol A set of rules to format and process data
Packet A small portion of a bigger file
Header Data at the beginning of a message summarizing
it's contents
Payload The size of a packet
Router A device that connects networks
Routing table A data file that has the network route
information

Problems in packet switched networks

  • Files can get lost or corrupted
  • Sometimes the computers overload and crash
  • Only some of the file gets through
  • During the class, we labeled the backs with to, from, and number of file. this helps the file get to the recipient easier.
  • We sent messages to ask who our neighbors neighbors are and etc. Then, we built a map. This helps us know where our message needs to go.
