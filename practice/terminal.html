How to open vim

  1. Open terminal. If you don't know how to get there, spotlight search with command+space, and type terminal
  2. cd to your website folder.
  3. Type vim index.html.
  4. Congrats, you've opened vim! Note, that index.html is the name of the file you created. Every time you say vim with your file name behind it, it will either open the existing file or make a new one depending on if you have a file of that or not.

The basics

To insert something into your file, you need to go into insert mode. This can be entered by pressing I on your keyboard. You can now freely type. To exit this mode, press esc.

The Minimal code

		The minimum code that you need for your file requires multiple things.
  • A declaration.
  • Not necessary, but you should have it-A language set
  • A head
  • A body
  • A closing tag for the html

How do I exit vim>

To exit and save changes in vim, type :wq. To exit vim without changes, type :q!. If you didn't make any changes, you can just type :q
